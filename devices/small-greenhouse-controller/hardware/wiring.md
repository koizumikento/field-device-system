# Wiring

## 配線の前提

- 制御ボックス: `Pico 2 WH + Robo Pico + MCP3008`
- サービスボックス: `8ch 出力ハブ + 12V 分配 + ポンプ6台`
- 温室内:
  - 上部センサーバー: `SHT41 + SCD41`
  - 上段: `U1 U2 U3`
  - 下段: `L1 L2 L3`
  - ファン: `上段 1台`、`下段 1台`
- 電源:
  - `5V系`: Pico / I2Cセンサー / MCP3008 / 出力ハブのロジック電源
  - `12V系`: ポンプ6台 / ファン2台
  - `GND` は共通にする

## 制御ボックス内の結線

### Pico 2 WH / Robo Pico と MCP3008

| 接続元 | 接続先 | 用途 |
| --- | --- | --- |
| `GP18` | `MCP3008 CLK` | SPI クロック |
| `GP19` | `MCP3008 DIN` | SPI MOSI |
| `GP16` | `MCP3008 DOUT` | SPI MISO |
| `GP17` | `MCP3008 CS/SHDN` | SPI CS |
| `3V3` | `MCP3008 VDD` | ADC ロジック電源 |
| `3V3` | `MCP3008 VREF` | ADC 基準電圧 |
| `GND` | `MCP3008 AGND` | アナログ GND |
| `GND` | `MCP3008 DGND` | デジタル GND |

### I2C バス

| 接続元 | 接続先 | 用途 |
| --- | --- | --- |
| `GP8` | `SHT41 SDA` | I2C SDA |
| `GP8` | `SCD41 SDA` | I2C SDA |
| `GP8` | `Conta STEMMA/Qwiic アダプター SDA -> 8ch 出力ハブ SDA` | I2C SDA |
| `GP9` | `SHT41 SCL` | I2C SCL |
| `GP9` | `SCD41 SCL` | I2C SCL |
| `GP9` | `Conta STEMMA/Qwiic アダプター SCL -> 8ch 出力ハブ SCL` | I2C SCL |
| `3V3` | `Conta STEMMA/Qwiic アダプター 3V3 -> 8ch 出力ハブ Vcc` | 出力ハブのロジック電源 |
| `GND` | `SHT41 GND / SCD41 GND / Conta STEMMA/Qwiic アダプター GND / 8ch 出力ハブ GND` | 共通 GND |

## 土壌水分センサーの結線

`SEN0308` は `3.3V - 5.5V` 駆動、出力は `0 - 3V` です。今回の構成では `MCP3008 VREF = 3.3V` に合わせ、6本とも `3.3V` で給電します。

| 鉢 | センサー信号 | MCP3008 |
| --- | --- | --- |
| `U1` | `AOUT` | `CH0` |
| `U2` | `AOUT` | `CH1` |
| `U3` | `AOUT` | `CH2` |
| `L1` | `AOUT` | `CH3` |
| `L2` | `AOUT` | `CH4` |
| `L3` | `AOUT` | `CH5` |

各センサーの共通配線は以下です。

| センサー線 | 接続先 |
| --- | --- |
| `VCC` | `3V3` |
| `GND` | `GND` |
| `AOUT` | 上表の `CH0-CH5` |

`CH6` と `CH7` は予備にして、将来のタンク液面や予備鉢に残します。

## サービスボックス内の結線

### 8ch 出力ハブと 12V 電源

| 接続元 | 接続先 | 用途 |
| --- | --- | --- |
| `12V 電源 +` | `8ch 出力ハブ V+` | 負荷電源の共通プラス |
| `12V 電源 -` | `8ch 出力ハブ GND` | 負荷側 GND |
| `制御ボックス GND` | `8ch 出力ハブ GND` | ロジック側との共通 GND |

### 出力チャネル割り当て

8ch 出力ハブは `0-7` を low-side switch として使います。各ポンプ/ファンの `+` 側は `12V+` にまとめ、`-` 側だけを出力ハブへ入れます。

| 出力チャネル | 接続先 |
| --- | --- |
| `0` | `Pump U1 -` |
| `1` | `Pump U2 -` |
| `2` | `Pump U3 -` |
| `3` | `Pump L1 -` |
| `4` | `Pump L2 -` |
| `5` | `Pump L3 -` |
| `6` | `上段ファン -` |
| `7` | `下段ファン -` |

以下は共通です。

| 負荷線 | 接続先 |
| --- | --- |
| `Pump U1-U3 / L1-L3 +` | `12V+ 共通` |
| `上段ファン +` | `12V+ 共通` |
| `下段ファン +` | `12V+ 共通` |

## 温室へのハーネス

### 上段ハーネス

上段は `U1-U3` と `上段ファン` をひと束にします。

| 束 | 中身 |
| --- | --- |
| `上段アナログ束` | `3V3`、`GND`、`U1_SIG`、`U2_SIG`、`U3_SIG` |
| `上段12V束` | `12V+`、`Pump U1-`、`Pump U2-`、`Pump U3-`、`Fan Upper-` |

### 下段ハーネス

下段は `L1-L3` と `下段ファン` をひと束にします。

| 束 | 中身 |
| --- | --- |
| `下段アナログ束` | `3V3`、`GND`、`L1_SIG`、`L2_SIG`、`L3_SIG` |
| `下段12V束` | `12V+`、`Pump L1-`、`Pump L2-`、`Pump L3-`、`Fan Lower-` |

### 上部センサーバー束

| 束 | 中身 |
| --- | --- |
| `I2C束` | `3V3`、`GND`、`SDA`、`SCL` |

`SHT41` と `SCD41` はこの `I2C束` から分岐させます。

## ラベル

保守のため、線とチューブは以下の名前でラベルします。

| 種別 | ラベル例 |
| --- | --- |
| 土壌水分センサー信号 | `MS-U1`, `MS-U2`, `MS-U3`, `MS-L1`, `MS-L2`, `MS-L3` |
| ポンプ出力 | `P-U1`, `P-U2`, `P-U3`, `P-L1`, `P-L2`, `P-L3` |
| ファン出力 | `F-UP`, `F-LOW` |
| I2C センサー束 | `I2C-BAR` |

## 組み方の優先順

1. `制御ボックス内` だけを先に組み、MCP3008 と I2C デバイスが見えるか確認する。
2. `サービスボックス内` を組み、出力 `0-7` を1本ずつ手動で ON/OFF して配線番号を確定する。
3. `上段ハーネス` を先に通し、`U1-U3` と `上段ファン` を結ぶ。
4. `下段ハーネス` を通し、`L1-L3` と `下段ファン` を結ぶ。
5. 最後に `ポンプチューブ` を入れて、配線束と分離して固定する。
