# Prototype Plan

## Phase 1

- `Pico 2 WH + Robo Pico + SHT41 + SCD41 + SEN0308 x1 + MCP3008 x1 + 8ch出力ハブ x1 + ポンプ x1` で机上試験を行う。
- MCP3008 の読み取り精度と出力ハブの I2C 制御を確認する。
- 12Vポンプの吐出量を秒単位で計測する。

## Phase 2

- 上段3鉢ぶんを先に組み、`上段ハーネス` の取り回しを固める。
- サービスボックスの位置と、チューブ3本束の固定方法を固める。
- `制御/センサ系 5V` と `アクチュエータ系 12V` の分離を確認する。

## Phase 3

- 下段3鉢を追加し、6鉢フル構成にする。
- 上段/下段ファンの風当たりと CO2 分布を確認する。
- Harvest Data 上で 6鉢の水分と給水イベントが追えることを確認する。
