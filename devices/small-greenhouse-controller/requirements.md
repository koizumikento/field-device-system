# Requirements

## Functional requirements

- 温室内の空気温度と湿度を定期計測できること。
- 温室内の CO2 を定期計測できること。
- 6鉢ぶんの土壌水分を個別に計測できること。
- 水分しきい値を下回った鉢だけを対象に給水できること。
- 同時給水を避け、1回に1鉢だけ給水できること。
- 設定した温度帯から外れたら送風または加温を切り替えられること。
- CO2 状態に応じて換気または循環を切り替えられること。
- センサー異常時は安全側に倒し、連続給水しないこと。
- センサーデータと主要イベントを `SORACOM Unified Endpoint` に送信し、`Harvest Data` へ蓄積できること。
- 通信断があってもローカル制御は継続できること。

## Constraints

- Power: `制御/センサ系 5V` と `アクチュエータ系 12V` は分離する。
- Wiring: センサー線とチューブは棚ごとの束にまとめ、扉の可動域を塞がない。
- Maintenance: ポンプ、チューブ、センサーを鉢番号単位で交換できること。
- Network: 通信経路は `Wi-Fi + SORACOM Arc` を前提にするが、通信断でもローカル制御を止めない。
- Installation environment: 高湿度、結露、散水飛沫、土汚れを前提にする。

## Assumptions

- 温室は `約幅50 x 奥行45 x 高さ130 cm` のスリム温室。
- 鉢は `上段3 + 下段3` に置く。
- 給水は液肥混入ではなく、まずは水のみ。

- 排気ファンを初期から入れるか。
