# Validation Checklist

## Bench checks

- 温湿度センサーが安定して値を返す。
- CO2 センサーが安定して値を返す。
- 6本の土壌水分センサーを `ADCハブ` で個別に読める。
- `8ch出力ハブ` からポンプ6台とファン2台を個別に制御できる。
- ポンプ駆動時にコントローラがリセットしない。
- `uni.soracom.io` への送信が成功する。

## Greenhouse checks

- 上段3鉢、下段3鉢の配線束が扉や植物に干渉しない。
- 給水後に対象鉢の水分値が期待方向へ変化する。
- 同時給水しない。
- センサー断線や異常値で連続給水しない。
- Harvest Data に 6鉢ぶんの水分と給水イベントが蓄積される。

## Acceptance for first usable prototype

- 24 時間以上、意図しない連続給水が起きない。
- 6鉢ぶんのセンサー値が継続取得できる。
- 少なくとも 3鉢以上で自動給水が安定する。
- Harvest Data 上で鉢ごとの差を追える。
