# outdoor-cloud-camera

## Summary

- What this device does: ソラカメを屋外に設置して、遠隔から現場の映像を確認する。
- Current status: `屋外スターターキットを使わない` 前提の初期 BOM、配線図、交換式バッテリー運用手順まで作成済み。現場条件に応じた固定方法と通信品質の最終確認が未了。
