# Requirements

## Functional requirements

- 屋外でカメラ映像を安定して取得できること
- `屋外スターターキットを使わず` に主要部材を個別調達できること
- 購入時に必要な主要部材と、現場依存の追加部材を分けて判断できること
- `7日交換 + 予備1本` の運用で止めずに回せること

## Constraints

- Power: `7日交換 + 予備1本` の交換式バッテリー運用を前提にし、`4W` 設計基準に対して `積雪 / 直射日光` 分の余裕を見て `ENEBOOST 12V-120Ah (LiFePO4)` を採用する
- Network: `+F FS045W + 楽天モバイル Rakuten最強プラン` を標準構成とし、APN は基本 `自動`、手動設定が必要な場合は `rakuten.jp` を使う
- Installation environment: 雨天、粉塵、屋外温度変化、`積雪`、`直射日光` を考慮し、防水ボックス、耐候固定材、日除け配慮を前提にする
- Cost: 現場依存の固定材は汎用品で吸収し、主要部材は公式購入品を優先する

## Fixed decisions

- カメラは `ATOM Cam Swing` を前提にする
- 1 現場 1 台構成から始める
- 通信は `+F FS045W + 楽天モバイル Rakuten最強プラン`
- 取付面は `外径φ48.6mm` の単管パイプ
- バッテリー箱と電装箱は `カメラ直下`
- バッテリーは `7日交換`
- 予備バッテリーは `1本`
- 遮熱は `白色アルミ複合板 300 x 300 x 3mm` を `30mm` 浮かせ、`電装箱とバッテリー箱の真上` に小庇として設置する
- 蓄電池の実型番は `ENEBOOST 12V-120Ah`
- 充電器の実型番は `BJ900F-12`
