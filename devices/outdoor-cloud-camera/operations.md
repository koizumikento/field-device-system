# Operations

## Setup

- 設置前に `既存 Wi-Fi が届くか` と `商用電源運用` か `交換式バッテリー運用` かを決める
- `既存 Wi-Fi` が不安定なら `+F FS045W` を追加する
- 取付面に応じて `1/4 インチネジ対応金具` と固定材を選ぶ
- カメラから給電点までの距離を測り、延長ケーブルと防雨収納の要否を決める
- バッテリー運用では [setup/battery-pack.md](./setup/battery-pack.md) と [diagrams/power-and-wiring.md](./diagrams/power-and-wiring.md) を基準に組む
- 取付方法は [setup/mounting-options.md](./setup/mounting-options.md) を基準に選ぶ

## Maintenance

- 定期的にレンズ面、防水ボックス、ケーブル固定、結露や浸水の有無を確認する
- 別調達ルーターを使う場合は、通信状態とルーター通電を点検する

## Weekly battery exchange

1. 現地到着前に `満充電済みの予備バッテリー` を用意する
2. ソラカメ映像が見えていることをスマホで確認する
3. バッテリー箱の `+/-` を確認し、一次側ヒューズを抜くか電源を遮断する
4. 使用中バッテリー箱を取り外し、満充電済みの予備バッテリー箱へ差し替える
5. ヒューズを戻し、`Blue Sea 1045` の USB 出力とカメラ起動を確認する
6. 映像復帰を確認してから現地を離れる
7. 取り外したバッテリーは持ち帰り、満充電後に次回交換用として保管する

## Weekly checks

- バッテリー交換時に `浸水`、`端子ゆるみ`、`ヒューズ溶断`、`ケーブル被覆傷` を確認する
- `RBM500` を入れている場合は、交換前の残容量を記録する
- 残容量が想定より速く落ちる場合は、`Wi-Fi再接続多発` や `低温` を疑う

## Failure handling

- 映像が見えない場合は、まずカメラ電源、`Wi-Fi` 到達性、別調達ルーター通電の順で切り分ける
- `+F FS045W` を使う場合は、`SIM認識`、`LTE接続表示`、`Wi-Fi SSID` を順に切り分ける
- 雨天後の不具合は、防水ボックスのシール状態とケーブル引き込み部を優先確認する
- バッテリー運用で起動しない場合は、`主ヒューズ`、`5V変換器`、`USBコネクタ抜け` の順で確認する

## Operational notes

- `ATOM Cam Swing` の同梱電源アダプタは防水非対応なので、屋外では屋内引き込みか防水ボックス収納が前提になる
