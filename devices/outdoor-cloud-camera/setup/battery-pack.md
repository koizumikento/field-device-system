# Battery Pack Setup

## 目的

- `ENEBOOST 12V-120Ah` を使った 7 日間運用の電源パックを組む
- 週次の `交換運用` を前提に、現地作業を短くする

## 使用部材

- [hardware/bom.md](../hardware/bom.md) の `具体 BOM 例` を使う

## 組み方

1. `BMO JAPAN バッテリーBOX 105A用 10D0009` に `ENEBOOST 12V-120Ah` を収納する。箱の内寸は `340 x 190 x 225mm`、採用バッテリーの外形は `329 x 170 x 216mm`、質量は `約14.5kg`
2. バッテリー `+端子` の近くに `Blue Sea 5061` と `10A ヒューズ` を入れる
3. バッテリー `+/-` から `BCPR203013SN` まで `1.25sq〜2sq` の 2 芯ケーブルを引く
4. バッテリー箱から出る DC 配線の末端に `Anderson Powerpole PP15/45` を付ける
5. `BCPR203013SN` 側にも対応する `Anderson Powerpole PP15/45` を付け、箱の内側で着脱できるようにする
6. `BCPR203013SN` に `Blue Sea 1045` を固定し、一次側を DC 配線へ接続する
7. カメラ側は `USBケーブル 4.5m` を `Blue Sea 1045` に接続する
8. `USBケーブル 4.5m` の箱出しは `Scanstrut DS16-P-BLK` を使って防水処理する
9. `FS045W` は `BCPR131508S` に収め、側面へ `PMF-6B` を付けて通気を確保する
10. バッテリー箱から電装箱までの DC 配線は、ふだんは常時接続し、週次交換時だけ電装箱を開けて `PP15/45` を切り離す
11. 電装箱を閉じる前に、`5V 出力`、`FS045W` 起動、カメラ起動を確認する
12. 起動確認後、バッテリー箱、電装箱、ルーター箱を現地へ固定する

## 交換運用の前提

- 現地には `電装箱 + カメラ` を残す
- `バッテリー箱` だけを外して持ち帰る
- `予備1本` を常備し、現地で入れ替えて停止時間を短くする
- 持ち帰ったバッテリーは `BJ900F-12` で満充電に戻す

## 交換しやすくする工夫

- バッテリー箱から電装箱へ行く配線は `短く、同じ向き` にそろえる
- `+` と `-` の識別を現地で迷わないようにラベルを付ける
- ヒューズ容量、交換日、満充電日を箱に記載する
- `PP15/45` は `電装箱の内側` に収め、雨天時に露出させない
- `ルーター箱` は `電装箱` のすぐ上か横へ寄せて、USB Type-C ケーブルを短く保つ

## 注意点

- `ATOM Cam Swing` の同梱 AC アダプタは使わない
- `5V / 2A` を安定供給できることを現地投入前に確認する
- 防水は `カメラ本体` だけでなく、`USB接続部` と `DC配線引き込み部` も対象にする
- `BMO JAPAN バッテリーBOX 105A用` は保護箱であって完全防水箱ではないので、上からの雨と直射は `小庇` で切る前提にする
- `積雪` と `直射日光` を考え、箱は地際を避け、`白色アルミ複合板 300 x 300 x 3mm` を `30mm` 浮かせた小庇の真下に置く

## LTE ルーターを追加する場合

- ルーターは `+F FS045W` を使う
- 給電は `Blue Sea 1045` の空き USB ポートから `FS045W` 同梱の `USB Type-C ケーブル` で行う
- SIM は `楽天モバイル Rakuten最強プラン` を使い、FS045W 側では `APN 自動` を基本、必要なら `rakuten.jp` を手動設定する
- 蓄電池は `ENEBOOST 12V-120Ah` を採用する
- `FS045W` は `BCPR131508S` へ収納し、`PMF-6B` を側面に入れる
- `FS045W` は設定用アプリと自動 APN 設定が使えるが、現地投入前に `SIM認識` と `自動起動` を確認する
- `FS045W` の動作温度範囲は `0°C〜35°C` なので、屋外では必ず `小庇 + 通風` を確保する
